IRINA'S COMPASS - Windows Setup
================================

1. Install Python (if not already installed)
   - Go to: https://www.python.org/downloads/
   - Download Python 3.11 or newer
   - IMPORTANT: During install, check "Add Python to PATH"

2. Run Irina's Compass
   - Double-click: START_Irinas_Compass.bat
   - First time: it will install dependencies (1-2 minutes)
   - Then your browser opens with the app

3. Bookmark the page
   - The app runs at: http://localhost:8501
   - You can bookmark it in your browser

4. To stop
   - Close the command window, or press Ctrl+C in it

TIPS FOR SEARCHING:
- Owner Name search: Type in Latin letters and it auto-converts to Georgian!
  Example: type "nana malenashvili" and it becomes "ნანა მალენაშვილი"
  You can edit the Georgian text before searching if needed.

Troubleshooting:
- If it says "python not found", reinstall Python and check "Add to PATH"
- If antivirus blocks the .bat file, allow it (it's just a launcher)
